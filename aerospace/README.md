# AeroSpace

タイル型ウィンドウマネージャー。

## Setup

```bash
ln -s ~/src/github.com/hiramatsutaku/dotfiles/aerospace/aerospace.toml ~/.aerospace.toml
```

## Keybindings

### フォーカス移動
- `alt-h/j/k/l` - 左/下/上/右

### ウィンドウ移動
- `alt-shift-h/j/k/l` - 左/下/上/右へ移動

### ワークスペース
- `alt-1~5` - ワークスペース切り替え
- `alt-shift-1~5` - ウィンドウをワークスペースに移動

### レイアウト
- `alt-/` - タイルレイアウト
- `alt-,` - アコーディオンレイアウト
- `alt-f` - フルスクリーン

### リサイズ
- `alt--` - 縮小
- `alt-=` - 拡大
