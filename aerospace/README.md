# AeroSpace

タイル型ウィンドウマネージャー。

## Setup

```bash
ln -s ~/src/github.com/hiramatsutaku/dotfiles/aerospace/aerospace.toml ~/.aerospace.toml
```

## Keybindings

### フォーカス移動 (Emacs-style)
- `cmd-b` - 左
- `cmd-f` - 右
- `cmd-p` - 上
- `cmd-n` - 下

### ウィンドウ移動
- `cmd-shift-b` - 左へ移動
- `cmd-shift-f` - 右へ移動
- `cmd-shift-p` - 上へ移動
- `cmd-shift-n` - 下へ移動

### ワークスペース
- `alt-1~9` - ワークスペース切り替え
- `alt-shift-1~9` - ウィンドウをワークスペースに移動
- `alt-tab` - 前のワークスペースに戻る
- `alt-shift-tab` - ワークスペースを次のモニターへ

### レイアウト
- `alt-/` - タイルレイアウト (水平/垂直切替)
- `alt-,` - アコーディオンレイアウト
- `alt-f` - フローティング/タイリング切替

### リサイズ
- `alt-shift--` - 縮小
- `alt-shift-=` - 拡大

### サービスモード (`alt-shift-;` で入る)
- `esc` - 設定リロード＆メインモードに戻る
- `r` - レイアウトリセット
- `f` - フローティング/タイリング切替
