# Zed

高速エディタ。

## Setup

```bash
ln -sf ~/src/github.com/hiramatsutaku/dotfiles/zed/settings.json ~/.config/zed/settings.json
ln -sf ~/src/github.com/hiramatsutaku/dotfiles/zed/keymap.json ~/.config/zed/keymap.json
```

## Keybindings

Emacs風キーバインド:
- `C-f/b/n/p` - カーソル移動
- `C-a/e` - 行頭/行末
- `M-f/b` - 単語移動
- `C-d/h` - 削除
- `C-k` - 行末まで削除
- `C-x C-b` - ファイル検索
- `C-x C-f` - サイドバー
- `C-x C-s` - 保存
- `C-enter` - 定義へジャンプ
