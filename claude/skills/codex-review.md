OpenAI Codexを使用してコードレビューを実行してください。

## 前提条件の確認
1. `which codex` でCodex CLIがインストールされているか確認
2. 未インストール → `npm install -g @openai/codex` を案内

## レビューの実行

未コミットの変更をレビュー:
```bash
codex exec "現在の未コミットの変更をレビューして、問題点や改善提案を日本語で報告してください"
```

## 出力フォーマット

### レビューサマリー
- 対象ファイル数
- 検出された問題の総数

### 問題点 (優先度順)
#### Critical (重大)
- [ファイル名:行番号] 問題の説明

#### Warning (警告)
- [ファイル名:行番号] 問題の説明

#### Suggestion (提案)
- [ファイル名:行番号] 改善の提案

### 総評
コード全体の品質と推奨アクション
