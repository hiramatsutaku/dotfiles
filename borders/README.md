# JankyBorders

macOS 用のウィンドウボーダー表示ツール。

## Setup

```bash
mkdir -p ~/.config/borders
ln -s ~/src/github.com/hiramatsutaku/dotfiles/borders/bordersrc ~/.config/borders/bordersrc
```

## Start Service

```bash
brew services start borders
```
